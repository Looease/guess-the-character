# Guess the  character

This is a full stack TypeScript express app.

Accessing:
The app uses pnpm as a package manager.

Clone the app.

To run the server: `cd server` `pnpm i` `pmpm dev`

To run the client: `cd client` `pnpm i` `pmpm start`
